# The Ecology of Decisions: Probability Distortion as Ecological Rationality

## Abstract

Bayesian decision theory is widely used as the normative benchmark for rational behaviour, yet human choices often deviate from Bayesian prescriptions in systematic ways. Rather than interpreting these deviations as cognitive biases, this project develops an ecological rationality framework in which behaviour is predicted directly from environmental structure.

We show that any monotonic decision policy can be represented as a threshold applied to a distortion of Bayesian posterior probabilities. Using a Prelec weighting function to parametrise this distortion, we model optimal decision-making under uncertainty relative to geometric mean fitness. Through large-scale simulations of flee–stay survival decisions across diverse ecological conditions, we demonstrate that the optimal probability distortion parameter is a predictable function of key ecological variables, primarily loss asymmetry and cue precision.

This introduces a framework for predicting behaviour from ecological parameters without fitting psychological constructs, and reframes probability distortion as a consequence of adaptation to environmental constraints.
